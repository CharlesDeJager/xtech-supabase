# Features

## Overview

This document outlines the main features of the **xtech-supabase** plugin.

## Core Features

- **Supabase Integration**  
   Seamless integration with Supabase for authentication, database, and storage.

- **Authentication**  
   Supports user sign-up, sign-in, and session management.

- **Database Operations**  
   CRUD operations for managing data with Supabase tables.

- **Real-time Updates**  
   Listen to and react to real-time changes in your data.

- **File Storage**  
   Upload, download, and manage files using Supabase Storage.

## Planned Features

- Additional data capabilites including data export
- Enhanced error handling
- Admin dashboard
- Additional supabase cli functions directly from extension
