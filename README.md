# XTECH Supabase

A VS Code extension for managing both local and linked Supabase environments directly from your editor.

## Documentation

- [Extension Details](docs/extension-details.md) - Full installation, configuration, authentication, and usage guide
- [Features](docs/features.md) - Feature-by-feature reference and capability matrix

## Quick Start

1. Install the extension from Marketplace or VSIX.
2. Open a workspace containing supabase/config.toml.
3. In VS Code Activity Bar, open Supabase Explorer.
4. Run XTECH Supabase: Refresh Supabase Explorer.

## Key Capabilities

- Dual environment explorer for Local and Linked
- Migration visibility and migration file creation
- Object introspection for schemas, tables, views, functions, triggers, enums, indexes, roles, policies, and storage buckets
- Inline object viewers for schema/data and object details

## Requirements

- VS Code 1.85.0+
- Supabase CLI installed and on PATH
- Supabase project workspace (supabase/config.toml)
